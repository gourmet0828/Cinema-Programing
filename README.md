# Cinema Program

## 프로젝트 소개
Cinema Program은 콘솔 환경에서 실행되는 영화관 예매 프로그램이다.  
사용자는 회원 또는 비회원으로 프로그램에 접속하여 영화 정보를 확인하고 좌석을 선택하여 예매할 수 있다.

프로그램은 영화 정보, 회원 정보, 예매 정보 등을 파일로 저장하여 관리하며, 로그인, 회원가입, 영화 예매, 예매 조회 및 취소 등의 기능을 제공한다.

---

## 개발 환경

Language : C  
OS : Windows 10 / Windows 11  
Execution : cmd (콘솔 환경)

프로그램은 컴파일된 실행 파일(.exe)을 통해 실행되며 Windows 환경의 cmd 창에서 정상 동작한다.

---

## 주요 기능

### 회원 기능
- 회원 가입
- 로그인
- 회원 정보 관리
- 포인트 사용

### 비회원 기능
- 전화번호와 생년월일을 이용한 예매 진행

### 영화 예매 기능
- 영화 선택
- 상영 시간 선택
- 예매 인원 입력
- 좌석 선택
- 결제 진행
- 쿠폰 사용
- 포인트 사용

### 예매 관리
- 예매 조회
- 예매 취소

### 로그 관리
- 프로그램 사용 기록 저장

---

## 프로그램 흐름

1. 프로그램 시작 후 현재 날짜와 시간을 입력한다.
2. 시작 메뉴에서 로그인 또는 회원가입을 선택한다.
3. 로그인 후 영화 예매 메뉴로 이동한다.
4. 영화 이름과 상영 시간을 선택한다.
5. 예매 인원과 좌석을 선택한다.
6. 쿠폰 또는 포인트 사용 후 결제를 진행한다.
7. 예매 완료 후 예매 조회 또는 취소가 가능하다.

## 데이터 파일

프로그램 실행 시 다음과 같은 데이터 파일이 생성되거나 사용된다.

Member.txt  
Guest.txt  
TheaterA.csv  
TheaterB.csv  
TheaterC.csv  
Reservation.csv  
DateTime.csv  
LogData.csv  

데이터 파일은 프로그램 실행 경로에 저장되며, 사용자가 직접 삭제하지 않는 이상 프로그램에서 자동으로 삭제되지 않는다.

---

## 실행 방법

1. 프로그램 실행 파일(main.exe)을 원하는 경로에 위치시킨다.
2. cmd 창을 실행한다.
3. 프로그램이 있는 폴더로 이동한다.

예시

cd C:\Users\User\Desktop\Cinema

4. 아래 명령어를 입력하여 프로그램을 실행한다.

main.exe
